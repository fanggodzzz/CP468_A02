INPUT FILE FORMAT: (graph text file)

*Problem use weighted non-directed graph
*Driver will start at node 0

- Notes:
    n: number of vertices
    m: number of edges
    p: number of parking lots
    in: number of invalid parking lots (farther than max walking distance)

    a1 a2 ... ap: list of parking lots nodes
    b1 b2 ... bp: list of invalid parking nodes
    u v w: edge (u, v) with weight w 

- Format:
n m p in
a1 a2 ... ap
b1 b2 ... bp
u1 v1 w1 
...
um vm wm

=====================================================================

OUTPUT FILE FORMAT: (output file after compiling "compare_performance.py")

- Used the csv format to extract values in excel:
    + First line is used for headers
    + Following lines is for data

=====================================================================
Notes: Compile in this order to get intended output file:
    training.py --> compare_performance.py

*COMPILING training.py:

- Use this command to compile:

    python training.py <graph_file> <episodes>
    current q_table.npy is the result of training 200000 episodes

*COMPILLING compare_performance.py:

- Use this command to compile:
    python compare_performance.py <q_file> <number of sample> <whether to print trials>

- Output file is "output.txt" in the csv form
